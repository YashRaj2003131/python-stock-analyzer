# python-stock-analyzer
A Python-based Stock Analyzer that uses real-time Yahoo Finance data to perform Company, Fundamental, Technical, Risk, CAPM, and Monte Carlo analyses. Built with Python, Pandas, NumPy, Matplotlib, and yFinance to support data-driven investment analysis.

# 📈 Python Stock Analyzer

A Python-based Stock Analyzer that provides comprehensive stock analysis using real-time market data from Yahoo Finance.

## 🚀 Features

- Company Information
- Fundamental Analysis
- Technical Analysis
- Risk Analysis
- CAPM (Capital Asset Pricing Model)
- Monte Carlo Simulation
- Real-time stock data using Yahoo Finance

## 📊 Analysis Modules

### 1. Company Information
- Company Name
- Sector
- Industry
- Current Price
- Market Cap
- Business Summary

### 2. Fundamental Analysis
- P/E Ratio
- EPS
- Dividend Yield
- Book Value
- ROE
- Debt-to-Equity Ratio
- 52-Week High & Low

### 3. Technical Analysis
- 50-Day Moving Average
- 200-Day Moving Average
- Bullish/Bearish Trend Detection

### 4. Risk Analysis
- Annual Return
- Variance
- Standard Deviation
- Annual Volatility
- Risk Classification

### 5. CAPM Analysis
- Beta
- Expected Return
- Alpha
- Market Sensitivity
- Valuation Status

### 6. Monte Carlo Simulation
- 1000 Stock Price Simulations
- Future Price Forecast
- Market Outlook

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/python-stock-analyzer.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python stock_analyzer.py
```
